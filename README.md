# AndHighlightSearch
A Chrome extension that supports matching multiple keywords simultaneously.
Here are the steps to install your custom Chrome extension:

---

# 🧩 How to Install a Custom Chrome Extension

## 1. Create a folder

Create a new folder on your computer, for example:

```
and-keyword-highlighter/
```

Put all your extension files inside it:

* manifest.json
* content.js
* popup.html
* popup.js
* background.js (if used)

---

## 2. Open Chrome Extensions page

Open Google Chrome and go to:

```
chrome://extensions/
```

---

## 3. Enable Developer Mode

In the top-right corner of the page:

👉 Turn ON **Developer mode**

---

## 4. Load the extension

Click the button:

👉 **“Load unpacked”**

Then select your extension folder:

```
and-keyword-highlighter/
```

---

## 5. Confirm installation

After loading:

* The extension will appear in the extensions list
* A new icon will show in the Chrome toolbar (top right area)

---

## 6. Pin the extension (recommended)

Click the puzzle icon 🧩 in Chrome toolbar, then:

👉 Pin your extension for quick access

---

## 7. Use the extension

1. Open any webpage
2. Click the extension icon
3. Enter keywords  For Ex: Apple,Red,Green
4. Click **“Start Highlight”**
5. Use:

   * Next
   * Previous
   * Match counter display (e.g. 3 / 18)

---

# ⚠️ If updates don’t apply

When you modify code:

1. Go to `chrome://extensions/`
2. Find your extension
3. Click **Reload (🔄)**

---

# 🎯 Done

Your custom AND keyword highlighter is now installed and ready to use.
